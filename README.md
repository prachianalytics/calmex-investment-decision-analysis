# calmex-investment-decision-analysis
Developed a capital budgeting decision model to evaluate alternative machinery investments using NPV and scenario analysis under changing cost conditions.
# Capital Budgeting NPV Analysis

## Overview
This project evaluates investment decisions for Calmex Company Ltd. using capital budgeting techniques.

## Key Concepts Used
- Net Present Value (NPV)
- Time Value of Money
- Scenario Analysis
- Investment Decision-Making
- Cash Flow Evaluation

## Objective
To compare two machinery investment options and recommend the best alternative based on profitability and risk evaluation.

## Tools Used
- Microsoft Excel
- Financial Modeling
- Corporate Finance Concepts

## Key Findings
- Option 1 generated higher NPV under both normal and changing cost conditions.
- NPV was selected as the most suitable capital budgeting method.
